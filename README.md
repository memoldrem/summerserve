Simple HTTP Server in C

This repository contains a basic HTTP server written in C. The server listens for incoming connections on a specified port and handles simple HTTP GET requests. This server demonstrates the use of sockets for network communication and basic HTTP response handling.

Features:
Listens on a specified port (default: 4221)
Handles basic HTTP GET requests
Responds with a 200 OK message for the root path /
Responds with a 404 Not Found message for any other path
